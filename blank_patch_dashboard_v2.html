Sheets-struktur

Appen forventer følgende faner i dit Google Sheet:

tracks — Track-oversigt med video-id og shorts-id
campaigns — Én row per kampagne
campaign_snapshots — Daglige snapshots per kampagne
channel_snapshots — Daglige kanal-snapshots
pitches — CRM med venue-status

Næste skridt

1. Importer den medfølgende .xlsx template til Google Drive (Fil → Importer)
2. Opret et Google Cloud projekt og aktiver Sheets API + YouTube Data API v3
3. Opret en API-nøgle og indsæt den herover
4. Kopier Sheets-ID fra URL'en på dit sheet (den lange kode mellem /d/ og /edit)
5. Gem og test forbindelsen
